# Roadmap

## Fase 0 - Descubrimiento UX
- Resultado: UX flows validados.
- Entregables: Wireframes Figma/Pen, user stories, risk matrix.
- Riesgos: Scope creep, API changes.

## Fase 1 - MVP frontend (1 mes)
- Resultado: CRUD básico funcional.
- Entregables: Dashboard/Users CRUD, API mock/integration, tests 80%.
- Riesgos: Integración backend delays.

## Fase 2 - Escalado y calidad (1 mes)
- Resultado: Production ready.
- Entregables: Perf opt, e2e tests, deploy pipeline, docs full.
- Riesgos: Regresiones UX.

## Dependencias
- Técnicas: Backend API endpoints listos.
- De negocio: User feedback loop.
